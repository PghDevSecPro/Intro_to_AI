CERT 6.2 Insider Threat — LSTM Autoencoder¶

Unsupervised anomaly detection on aggregated user-day behavioural features.

Approach¶

  • Training: fit the autoencoder on baseline (non-malicious) user sequences only
  • Detection: flag users whose daily behaviour has unusually high reconstruction error
  • Input: fixed-length sliding windows of user behavioural features (logon/logoff, device, file, email, HTTP activity, temporal patterns)

5. Sequence Construction¶

Each user's chronologically sorted feature rows become sliding windows of SEQUENCE_LENGTH days. A window is labelled malicious (1) if it contains at least one malicious day, benign (0) otherwise.

  • Training: non-overlapping stride keeps the training set tractable (~35 k sequences for 3 200 users × ~350 days).
  • Test: stride = 1 ensures every malicious day is captured in at least one window.
In [9]:
def build_sequences(df, feature_cols, seq_len, stride, label_col="is_malicious"):
    
    X_list, y_list, meta_list = [], [], []

    for user_id, group in df.groupby("user_id"):
        group = group.sort_values("day").reset_index(drop=True)
        n = len(group)
        if n < seq_len:
            continue

        feats  = group[feature_cols].values.astype(np.float32)
        labels = (
            group[label_col].values.astype(int)
            if label_col in group.columns
            else np.zeros(n, dtype=int)
        )
        days = group["day"].values

        for start in range(0, n - seq_len + 1, stride):
            end = start + seq_len
            X_list.append(feats[start:end])
            y_list.append(int(labels[start:end].any()))
            meta_list.append({"user_id": user_id, "last_day": days[end - 1]})

    X = np.array(X_list, dtype=np.float32)
    y = np.array(y_list, dtype=int)
    return X, y, meta_list

6. LSTM Autoencoder¶

class LSTMAutoencoder(nn.Module):


    def __init__(
        self,
        input_size: int,
        hidden_size: int = 64,
        num_layers: int = 2,
        dropout: float = 0.1,
    ):
        super().__init__()
        self.hidden_size = hidden_size
        self.num_layers  = num_layers

        enc_dropout = dropout if num_layers > 1 else 0.0
        dec_dropout = dropout if num_layers > 1 else 0.0

        self.encoder = nn.LSTM(
            input_size, hidden_size, num_layers,
            batch_first=True, dropout=enc_dropout,
        )
        self.decoder = nn.LSTM(
            hidden_size, hidden_size, num_layers,
            batch_first=True, dropout=dec_dropout,
        )
        self.output_layer = nn.Linear(hidden_size, input_size)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        
        seq_len = x.size(1)

        # Encode: 
        _, (h_n, c_n) = self.encoder(x)                         

        # Tile the final encoder hidden state
        context = h_n[-1].unsqueeze(1).repeat(1, seq_len, 1)    

        # Decode:
        decoded, _ = self.decoder(context, (h_n, c_n))           

    
        reconstructed = self.output_layer(decoded)               
        return reconstructed


model = LSTMAutoencoder(
    input_size=n_features,

8. Anomaly Detection¶

Anomaly threshold = THRESHOLD_PERCENTILE-th percentile of training reconstruction errors. Any test window whose MSE exceeds this threshold is flagged as anomalous.

In [13]:
def compute_reconstruction_errors(model, loader, device):
    
    model.eval()
    errors = []

    # Generative AI assisted writing this function:
    with torch.no_grad():
        for batch in loader:
            batch_x = batch[0].to(device)
            recon   = model(batch_x)
            # Mean over (seq_len, n_features) -> one scalar per sequence
            mse = ((recon - batch_x) ** 2).mean(dim=(1, 2))
            errors.append(mse.cpu().numpy())
    # --------------------------------------------


    return np.concatenate(errors)


# Training-set errors / anomaly threshold
